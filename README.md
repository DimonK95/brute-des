# brute-des
Multithreading brute-force DES-encrypted password cracker.

Command Line Parameters:

    i - iterative brute mode 
    r - recursive brute mode
    n - max length of password 
    s - single-thread mode
    m - multi-thread mode
    a - alphabet

Default settings:

    brute mode is iter
    alphabet is "abc"
    max length is 4
    thread mode is single
